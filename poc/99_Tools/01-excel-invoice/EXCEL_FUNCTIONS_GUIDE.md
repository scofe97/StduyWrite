# 거래명세표에 사용된 엑셀 수식 완전 가이드

> 엑셀을 처음 접하는 사람도 이해할 수 있도록 하나씩 설명합니다.

---

## 엑셀 수식의 기본 규칙

엑셀에서 수식은 항상 **`=` (등호)** 로 시작한다.

```
= 1 + 2       → 결과: 3
= A1 + B1     → A1 셀의 값과 B1 셀의 값을 더한다
```

`A1`, `B3` 같은 것은 **셀 주소**다. 엑셀은 열(A, B, C...)과 행(1, 2, 3...)의 조합으로 각 칸을 부른다.

---

## 1. VLOOKUP — "코드번호로 품명 찾기"

### 이 수식이 하는 일

코드번호 `40040036401`을 입력하면, 코드매핑 시트에서 해당 코드를 찾아서 품명 `BEARING`을 가져온다.
사전에서 단어를 찾으면 뜻이 나오는 것과 같다.

### 실제 사용된 수식

```
=VLOOKUP(B9, 코드매핑!$A:$E, 2, FALSE)
```

### 한 조각씩 뜯어보기

| 조각 | 의미 | 비유 |
|------|------|------|
| `VLOOKUP(` | "세로로 찾아라" (Vertical Lookup) | 사전을 위에서 아래로 훑는 것 |
| `B9` | 찾을 값이 있는 셀 (코드번호) | "이 단어를 찾아줘" |
| `코드매핑!$A:$E` | 찾을 범위 (코드매핑 시트의 A~E열) | "이 사전에서 찾아" |
| `2` | 몇 번째 열의 값을 가져올지 (2 = 품명) | "뜻은 두 번째 칸에 있어" |
| `FALSE` | 정확히 일치하는 것만 찾기 | "비슷한 것 말고 정확한 것만" |

### 열 번호에 따라 가져오는 값이 다르다

코드매핑 시트 구조:

```
열:  A(1열)      B(2열)    C(3열)     D(4열)   E(5열)
     코드번호     품명      규격       단가     비고
     40040036401  BEARING   RNA6906    5800     112DT PUMP
```

| 수식의 열 번호 | 가져오는 값 | 예시 결과 |
|---------------|-----------|----------|
| `2` | 품명 | BEARING |
| `3` | 규격 | RNA6906 |
| `4` | 단가 | 5,800 |
| `5` | 비고 | 112DT PUMP |

### `$` 기호의 의미

`$A:$E`에서 `$`는 **"이 범위를 고정해"** 라는 뜻이다.
수식을 다른 행에 복사해도 항상 A~E열을 참조한다. (절대 참조)

```
$A:$E  → 항상 A~E열 고정 (아래로 복사해도 변하지 않음)
A:E    → 복사하면 B~F, C~G로 밀림 (상대 참조)
```

## 2. IFERROR — "에러가 나면 빈칸으로"

### 이 수식이 하는 일

코드번호를 아직 입력하지 않았을 때, VLOOKUP이 찾을 값이 없어서 `#N/A` 에러가 뜬다.
IFERROR는 이 에러를 감지해서 빈 칸(`""`)으로 바꿔준다.

### 실제 사용된 수식

```
=IFERROR(VLOOKUP(B9, 코드매핑!$A:$E, 2, FALSE), "")
```

### 구조

```
=IFERROR(  원래 수식  ,  에러일 때 표시할 값  )
```

| 조각 | 의미 |
|------|------|
| `IFERROR(` | "에러가 나면 대신 이걸 보여줘" |
| `VLOOKUP(...)` | 원래 실행할 수식 |
| `""` | 에러 시 빈 칸 표시 (빈 문자열) |

### 있을 때 vs 없을 때

```
B9에 코드번호가 있을 때:
  VLOOKUP 성공 → "BEARING" 표시

B9이 비어있을 때:
  VLOOKUP 실패 → #N/A 에러 발생 → IFERROR가 잡아서 → "" (빈 칸) 표시
```

---

## 3. IF — "조건에 따라 다르게"

### 이 수식이 하는 일

코드번호가 입력되었으면 "EA"를 표시하고, 비어있으면 아무것도 표시하지 않는다.

### 실제 사용된 수식

```
=IF(B9="", "", "EA")
```

### 구조

```
=IF(  조건  ,  참일 때  ,  거짓일 때  )
```

| 조각 | 의미 |
|------|------|
| `IF(` | "만약에..." |
| `B9=""` | "B9 셀이 비어있으면" |
| `""` | 참이면 → 빈 칸 |
| `"EA"` | 거짓이면(= 뭔가 입력됨) → "EA" 표시 |

### 흐름도

```
B9이 비어있나?
  ├── YES → 빈 칸 표시
  └── NO  → "EA" 표시
```

## 4. AND — "두 조건 모두 만족할 때"

### 이 수식이 하는 일

공급가액 계산에서 수량과 단가가 **둘 다** 있을 때만 곱셈을 한다.
하나라도 비어있으면 빈 칸으로 둔다.

### 실제 사용된 수식

```
=IF(AND(E9<>"", G9<>""), E9*G9, "")
```

### 한 조각씩

| 조각 | 의미 |
|------|------|
| `AND(E9<>"", G9<>"")` | E9(수량)이 비어있지 않고, G9(단가)도 비어있지 않을 때 |
| `E9*G9` | 수량 × 단가 계산 |
| `""` | 둘 중 하나라도 비면 빈 칸 |

> `<>` 는 "같지 않다"는 뜻이다. `E9<>""` = "E9이 빈 칸이 아니다"

### 흐름도

```
수량(E9)이 있고 AND 단가(G9)도 있나?
  ├── YES → E9 × G9 = 공급가액
  └── NO  → 빈 칸
```

## 5. SUM — "범위의 합계"

### 이 수식이 하는 일

9행부터 17행까지의 공급가액을 모두 더한다.

### 실제 사용된 수식

```
=SUM(H9:H17)
```

### 구조

```
=SUM(  시작셀 : 끝셀  )
```

| 조각 | 의미 |
|------|------|
| `SUM(` | "다 더해라" |
| `H9:H17` | H9부터 H17까지 (9개 행) |

이것은 아래와 같다:

```
= H9 + H10 + H11 + H12 + H13 + H14 + H15 + H16 + H17
```

빈 셀이나 텍스트는 자동으로 무시된다.

## 6. INT — "소수점 버리기"

### 이 수식이 하는 일

세액 계산에서 소수점 이하를 버린다. (원 단위 절사)

### 실제 사용된 수식

```
=INT(H18 * 0.1)
```

| 조각 | 의미 |
|------|------|
| `H18 * 0.1` | 공급가액 합계의 10% |
| `INT(...)` | 소수점 이하를 버림 |

### 예시

```
공급가액 합계 = 123,456원
세액 = INT(123456 × 0.1) = INT(12345.6) = 12,345원
```

---

## 7. 산술 수식 — "더하기, 곱하기"

### 공급가액 (각 행)

```
= E9 * G9     → 수량 × 단가
```

### 합계 (원정)

```
= H18 + H19   → 공급가액 합계 + 세액
```

| 연산자 | 의미 | 예시 |
|--------|------|------|
| `+` | 더하기 | `=A1+B1` |
| `-` | 빼기 | `=A1-B1` |
| `*` | 곱하기 | `=A1*B1` |
| `/` | 나누기 | `=A1/B1` |

## 8. 데이터 유효성 검사 — "드롭다운 목록"

이것은 수식이 아니라 **엑셀 기능**이다.

### 하는 일

코드번호 셀(B9~B17)을 클릭하면 드롭다운 화살표가 나타나고,
코드매핑 시트에 등록된 221개 코드번호 중에서 선택할 수 있다.

### 동작 방식

```
B9 셀 클릭 → ▼ 화살표 표시 → 클릭하면 목록 펼침 → 코드 선택
```

잘못된 코드를 직접 타이핑하면 "코드번호 오류" 경고가 뜬다.

### 직접 만들어보려면

1. 셀 선택
2. 메뉴: 데이터 → 데이터 유효성 검사
3. 허용: 목록
4. 원본: `코드매핑!$A$2:$A$222`

## 전체 수식 흐름도

```
[사용자 입력]              [자동 계산]                [최종 결과]

코드번호 선택 ──→ VLOOKUP ──→ 품명, 규격, 단가, 비고
  (B9)              │
                    │         IF ──→ 단위 "EA"
                    │
수량 입력 ────────────┘
  (E9)              │
                    ▼
              수량 × 단가 ──→ 공급가액 (H9)
                               │
                               ▼
                           SUM(H9:H17) ──→ 공급가액 합계 (H18)
                               │
                               ▼
                        INT(합계×0.1) ──→ 세액 (H19)
                               │
                               ▼
                        합계 + 세액 ──→ 합계 원정 (H20)
```

## 수식 요약 카드

| 수식 | 한줄 설명 | 이 파일에서의 역할 |
|------|----------|------------------|
| `VLOOKUP` | 표에서 값을 찾아온다 | 코드번호 → 품명/규격/단가/비고 |
| `IFERROR` | 에러를 다른 값으로 바꾼다 | 코드 미입력 시 빈 칸 표시 |
| `IF` | 조건에 따라 다른 값 표시 | 코드 있을 때만 "EA" 표시 |
| `AND` | 여러 조건이 모두 참인지 확인 | 수량+단가 둘 다 있을 때만 계산 |
| `SUM` | 범위의 합계 | 공급가액 합계 |
| `INT` | 소수점 버림 | 세액 원 단위 절사 |
| `*`, `+` | 곱하기, 더하기 | 공급가액, 총합계 |
