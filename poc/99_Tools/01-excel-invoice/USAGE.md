# 거래명세표 Excel 사용 가이드

## 개요

베어링/부품 거래명세표 자동화 엑셀 파일.
코드번호 선택 시 품명/규격/단가가 자동 매핑되고, 수량 입력 시 공급가액이 자동 계산된다.

- 공급받는자: ㈜제일피엠씨 (603-81-70100)
- 공급자: ㈜동명베아링 (605-81-10092)

## 사용법

### 1. 파일 열기
`거래명세표.xlsx`를 Excel로 연다. (매크로 없음, 보안 설정 불필요)

### 2. 날짜 입력
Row 2의 날짜 셀에 거래 날짜를 직접 입력한다.

### 3. 코드번호 선택 (노란색 셀)
- B9~B17 셀 (노란색 배경) 클릭
- 드롭다운 화살표에서 코드번호 선택
- 또는 코드번호를 직접 입력 (예: `40040036401`)

### 4. 자동 매핑 확인
코드번호 입력 시 다음이 자동으로 채워진다:
| 열 | 내용 | 수식 |
|----|------|------|
| 품명 | BEARING, OIL SEAL 등 | VLOOKUP |
| 규격 | RNA6906, 6200ZZ 등 | VLOOKUP |
| 단가 | 숫자 (콤마 포맷) | VLOOKUP |
| 단위 | EA (자동) | IF |
| 비고 | 용도 (112DT PUMP 등) | VLOOKUP |

### 5. 수량 입력 (노란색 셀)
E9~E17 셀 (노란색 배경)에 수량을 입력한다.

### 6. 자동 계산 확인
- **공급가액** = 수량 x 단가 (각 행)
- **공급가액 합계** = SUM (Row 18)
- **세액** = 공급가액 합계 x 10% (Row 19)
- **합계(원정)** = 공급가액 + 세액 (Row 20)

### 7. 인쇄
- 인쇄 영역이 A4 가로로 설정되어 있음
- `Ctrl+P`로 바로 인쇄 가능

## 시트 구성

| 시트 | 설명 |
|------|------|
| 거래명세표 | 메인 양식 (입력/출력) |
| 코드매핑 | 조회 테이블 (221개 품목) |

## 등록 품목 (221개)

| 분류 | 수량 | 코드 접두사 | 예시 규격 |
|------|------|------------|----------|
| BEARING | 122종 | 4004xxxx | 6000~6300 시리즈, NUP/NU/NJ, RNA/NK/HK, 자동조심, 앵귤러, 테이퍼 |
| OIL SEAL | 30종 | 5504xxxx | TC/SC/VC 타입 |
| COLLAR | 20종 | 3304xxxx | IR 시리즈 이너링 |
| STEEL BALL | 10종 | 4004xxxx | 1/4"~1-1/4" |
| O-RING | 12종 | 6604xxxx | P/G/S 시리즈 |
| BUSH | 10종 | 7704xxxx | DU 시리즈 |

## 품목 추가/수정

코드매핑 시트에서 직접 행을 추가/수정하면 된다.
드롭다운 목록은 코드매핑 시트 A열 전체를 참조하므로 자동 반영된다.

## 재생성

품목 데이터를 대량 변경하려면 `create_invoice.py`의 `get_code_mapping_data()` 함수를 수정 후 재실행:

```bash
python3 create_invoice.py
```

> 주의: 재실행 시 기존 `거래명세표.xlsx`가 덮어씌워진다.
