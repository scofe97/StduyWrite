# 11. 클로저와 함수형 패턴

클로저는 환경을 캡처할 수 있는 익명 함수입니다. Rust는 캡처 방식에 따라 Fn, FnMut, FnOnce 세 가지 트레이트로 클로저를 분류하며, 컴파일러가 자동으로 가장 적절한 트레이트를 추론합니다. Java 람다와 달리 명시적 move와 소유권 시스템이 결합되어 메모리 안전성을 보장합니다.

## 명령어 요약
| 개념 | 설명 |
|------|------|
| `\|x\| x + 1` | 클로저 기본 문법 |
| `Fn` | 불변 빌림으로 캡처, 여러 번 호출 가능 |
| `FnMut` | 가변 빌림으로 캡처, 내부 상태 변경 가능 |
| `FnOnce` | 소유권 이동, 한 번만 호출 가능 |
| `move` | 캡처한 변수의 소유권을 클로저로 이동 |
| `impl Fn(i32) -> i32` | 클로저 반환 타입 |
