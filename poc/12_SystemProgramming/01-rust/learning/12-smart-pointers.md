# 12. 스마트 포인터

스마트 포인터는 포인터처럼 동작하지만 메타데이터와 추가 기능을 가진 데이터 구조입니다. Rust의 대표적 스마트 포인터인 Box, Rc, Arc, RefCell은 각각 힙 할당, 참조 카운팅, 스레드 안전 카운팅, 내부 가변성을 제공합니다. Java의 GC와 달리 컴파일 타임에 소유권과 Drop 트레이트로 메모리를 관리합니다.

## 명령어 요약
| 개념 | 설명 |
|------|------|
| `Box<T>` | 힙 할당, 소유권 이동 |
| `Deref` | `*` 연산자 오버로딩 |
| `Drop` | 소멸자, 자동 메모리 해제 |
| `Rc<T>` | 단일 스레드 참조 카운팅 |
| `Arc<T>` | 멀티 스레드 참조 카운팅 (Atomic) |
| `RefCell<T>` | 런타임 빌림 검사, 내부 가변성 |
| `Weak<T>` | 참조 순환 방지, 카운트 증가 없음 |
