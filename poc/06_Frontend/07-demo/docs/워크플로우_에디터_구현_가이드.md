# IRIS 워크플로우 에디터 구현 가이드

## 📋 개요

> **참고**: 이 프로젝트(`React-demo`)는 원래 **shadcn/ui의 네이티브 코드에서 추출한 디자인 패턴과 컴포넌트 설계 규칙을 학습**하기 위한 프로젝트입니다. 워크플로우 에디터는 학습한 패턴을 실제로 적용해보는 **실습 예제**로 추가되었습니다.

React Flow 기반의 노드 워크플로우 에디터를 구현한 과정을 정리한 문서입니다.
IRIS 스타일의 다크 테마 UI와 드래그 앤 드롭 기능을 갖춘 워크플로우 빌더입니다.

## 🛠️ 기술 스택

## 🎨 주요 컴포넌트

### 1. Layout.tsx - 공통 레이아웃

상단 네비게이션 바를 모든 페이지에서 표시합니다.

```tsx
// 주요 기능
- IRIS 로고 (그라데이션)
- 메뉴: 워크플로우 / 이미지 / 비디오 / 템플릿 / 일괄 작업
- 활성 메뉴 표시 (activeMenu prop)
- 고정 위치 (fixed top)
```

### 2. WorkflowDashboard.tsx - 대시보드

워크플로우 목록을 카드/리스트 형태로 표시합니다.

```tsx
// 주요 기능
- 검색 필터
- 상태 필터 (전체/초안/활성/일시정지/보관함)
- 그리드/리스트 뷰 토글
- 워크플로우 카드 (상태 배지, 실행 통계)
- 새 워크플로우 생성 버튼
```

### 3. WorkflowCanvas.tsx - 에디터 캔버스

React Flow 기반의 노드 에디터입니다.

```tsx
// 주요 기능
- 서브헤더 (뒤로가기, 워크플로우 이름, 검증/저장/실행 버튼)
- 노드 드래그 앤 드롭
- 노드 간 연결 (점선 애니메이션)
- 미니맵, 줌 컨트롤
- 배경 도트 그리드
```

### 4. NodePalette.tsx - 노드 팔레트

왼쪽 사이드바의 노드 목록입니다.

```tsx
// 주요 기능
- 노드 검색
- 카테고리별 접기/펼치기 (생성기, 분석기, 편집기, 유틸리티)
- 드래그하여 캔버스에 추가
```

### 5. 커스텀 노드들

#### TriggerNode (이벤트 트리거)
- 색상: 에메랄드 그린
- 출력: Event Data

#### SplitNode (분할)
- 색상: 그레이
- 입력: Input (필수)
- 출력: Parts

#### ImageGenNode (이미지 생성)
- 색상: 퍼플 (보라색 테두리)
- 입력: Prompt*, Negative Prompt, Reference Image
- 출력: Generated Image

## 🔧 핵심 구현 코드

### 노드 타입 등록

```tsx
// nodes/index.ts
export const nodeTypes = {
  trigger: TriggerNode,

  imageGen: ImageGenNode,
};
```

### 드래그 앤 드롭 구현

```tsx
// NodePalette.tsx - 드래그 시작
const onDragStart = (event: DragEvent, nodeType: string, label: string) => {
  event.dataTransfer.setData('application/reactflow/type', nodeType);
  event.dataTransfer.setData('application/reactflow/label', label);
  event.dataTransfer.effectAllowed = 'move';
};

// WorkflowCanvas.tsx - 드롭 처리
const onDrop = (event: DragEvent) => {
  const type = event.dataTransfer.getData('application/reactflow/type');
  const label = event.dataTransfer.getData('application/reactflow/label');
  
  const position = screenToFlowPosition({
    x: event.clientX,
    y: event.clientY,
  });
  
  const newNode = { id: `${type}-${nodeId++}`, type, position, data: { label } };
  setNodes((nds) => [...nds, newNode]);
};
```

### 엣지 스타일 (점선 애니메이션)

```tsx
const edgeStyle = {
  stroke: '#a855f7',      // 퍼플
  strokeWidth: 2,
  strokeDasharray: '5,5', // 점선
};

// 연결 시 적용
onConnect={(connection) => {
  addEdge({ ...connection, animated: true, style: edgeStyle }, edges);
}}
```
