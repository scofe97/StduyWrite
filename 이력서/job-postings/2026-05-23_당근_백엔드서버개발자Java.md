## 도메인 한 줄 + 예상 기술부채

**도메인**: 지역 기반 중고 직거래 + 동네 생활·커머스 슈퍼앱 "당근". MAU 1000만·하루 평균 20분 사용·총 480억 원 투자 유치 단계의 신사업 확장기.

**예상 기술부채 3개**
1. **결제 플랫폼·커머스 플랫폼·3rd-party 플랫폼 동시 진행** — 세 플랫폼이 병행되면서 공통 트랜잭션 경계·인증·재시도 정책이 흩어졌을 가능성. 외부 호출 단일 추상화 + 멱등 패턴 도입 가능
2. **신사업 다수 진행으로 도메인 SSOT 미정착** — "신사업 Product 개발"이 주요 업무 = 카탈로그·계약·정산 데이터가 신사업마다 따로 모델링됐을 가능성. CDC + 도메인 이벤트로 정합성 보강 가능
3. **MAU 1000만 트래픽 압력 + MSA 분리 진행 중** — 자격 요건의 백오피스 자체 제작 강조 = 마이크로서비스 분리가 일부만 끝났을 신호. 분산 트랜잭션 SAGA·Outbox 패턴 적용 가능
