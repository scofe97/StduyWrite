## 매칭 분석 (vs RESUME.md)

**강점 (이 공고에서 어필 가능한 내 마일스톤)**
- **Kubernetes 운영 + CKA**: 개인 GCP 3노드 K8s 자가호스팅 → 필수 직접 매칭
- **Helm 운영**: RESUME §3 신규 카드 "Jenkins 컨트롤러를 Helm 차트로 K8s 배포" + Side Project Helm 운영 → 필수 정확 매칭
- **CI/CD 파이프라인**: Jenkins 멀티 인스턴스 표준화 + Job 이관 파이프라인 자동화 → 필수 매칭
- **관찰성 스택(Grafana)**: 개인 GCP 클러스터에 Grafana·Loki·Tempo OpenTelemetry 자가 관찰성 구성 → 필수 Grafana 직접 매칭
- **클라우드 보안 컴플라이언스 사고**: 금융권 변경관리·감사 추적 + 결재선 자동화 → "보안 컴플라이언스 관리" 사고 회로 정합
- **AI 도메인 정합**: IATA 차세대 항공권 유통 기술 인증 + AI 그랜드 챌린지 장관상 = AI 도메인. 본인 Claude Code·OMC 운영 경험으로 우대 보강 가능

**공백**
- **AWS 상용 운영 경험 부재** (§14 누적 3순위, 자격증만)
- **AWS CDK·CloudFormation 직접 작성 경험 없음** — 본인은 GCP + 수기 kubeadm. AWS CDK는 TypeScript 기반이라 §14 누적 JVM 외 언어 공백과 결합. 학습 또는 우회 답변 준비
- **Terraform 직접 작성 경험 부족** — 비댁스와 동일 공백
- **NewRelic·Datadog·Sentry 직접 운영 경험 없음** — Grafana·Loki·Tempo OSS 스택만 운영. 상용 APM 학습 또는 OSS 운영 경험을 "관찰성 사고 회로" 일반화로 답변
