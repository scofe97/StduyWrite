## 도메인 한 줄 + 예상 기술부채

**도메인**: 글로벌 K-Pop 이커머스 플랫폼 "ktown4u". 200개국 이상·500만 회원·매년 2배 이상 빠른 성장 단계.

**예상 기술부채 3개**
1. **200개국 글로벌 트래픽 + 지역별 결제·배송 규제 분산** — 글로벌 명시 = 다국 통화·세금·배송 규제가 코드에 분산됐을 가능성. 외부 호출 단일 추상화 + 지역별 어댑터 분리 패턴 적용 가능
2. **MSA 전환 + Kafka 비동기 흐름 정착 단계** — 필수 Kafka 명시 = 이벤트 기반 분리 진행 중. 주문·재고·정산의 분산 트랜잭션 SAGA·Outbox 도입 가능
3. **AI Agent 도입 초기 단계** — 우대 "AI 도구·AI Agent 활용" = AI 통합 인프라(임베딩 캐싱·LLM 호출 추상화·비용 관리) 아직 정착 안 됨. 본인 Claude Code 운영 패턴 직접 적용 가능
