## 3. 기존 14건 분석 대비 2026 변화 (검증 결과)

| 기존 공백 | 2026 검증 | 근거 |
|---|---|---|
| **JVM외 언어(Kotlin/Node/TS)** | **우선순위 하향** | 2026 시장은 언어 폭보다 깊은 엔지니어링을 보상. Java 여전히 고수요. 공백은 맞으나 시급도 낮음 |
| **대용량 트래픽 정량 메트릭** | **유효, 시급도 상승** | seniority-bias 심화로 정량 증거가 시니어 전환 게이트 |
| **IaC(Terraform)** | **유효, 최우선 격상** | 2026 "baseline 스킬". 없으면 감점. Sonatus·Binance 직접 요구 |
| **AWS 상용운영(자격증만)** | **유효** | 멀티클라우드·FinOps 전제. 다수 공고 hands-on 요구 |
| **Redis 운영** | **부분 유효, 시급도 하향** | 단독 공백 신호 약함. 캐시/세션 일반 역량으로 묶임 |

**2026 신규 공백**: FinOps(별도 직무 부상), eBPF(Cilium/Tetragon 차세대 관측성).

## 4. 2026 신규 부상 요구 (전부 실재 확인)

- **플랫폼 엔지니어링/IDP**: Gartner 80% 조직 전담 플랫폼팀 예측(2026). 표준 "BACK 스택" = Backstage+ArgoCD+Crossplane+Kyverno. DevSecOps/플랫폼 직무 10~20% 연봉 프리미엄.
- **AI/LLM 활용 개발**: 2026 SWE 95% 주1회+ AI 사용, 75% 업무 절반 이상. Claude Code 사실상 표준, MCP로 내부 도구 노출이 일상.
- **eBPF**: Cilium 중심 차세대 관측성 표준(Pixie/Parca/Tetragon).
- **FinOps**: FinOps Cloud Engineer 별도 직무 부상.
- **DORA/DevEx + AI 리터러시**: platformengineering.org가 2026 "생존 수준 역량"으로 명시.

### Claude Code/MCP 경험 어필 가능성 (검증)
- **증가 확인**: Staff AI Engineer·Agentic AI Developer 공고가 "MCP 서버 개발/배포, agentic workflow 설계"를 명시 요구(미국·AI 전담).
- **한계(추측)**: 한국 일반 백엔드 공고가 MCP를 필수로 거는 사례는 미확인 — "AI 도구 능숙" 우대 수준에 머물 가능성. MCP 명시는 AI 플랫폼/스타트업 한정으로 보임.
- **이원 전략**: 일반 백엔드엔 "AI 에이전트로 생산성" 우대 어필, AI 플랫폼/DevEx 직무엔 핵심 차별화로 포지셔닝.
