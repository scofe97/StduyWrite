# 05_JVM ch02 메모리 영역 학습 오답 노트

> §14 자가평가에서 점수 낮은 자리 + 4-Phase 검증 자답에서 막히거나 어긋난 자리를 박제한다. 같은 패턴이 3회 이상 반복되면 *그 챕터 자체가 약하다*는 신호이므로 본문 보강 트리거로 본다.

## 패턴 1 — 답을 "처방"으로 바로 점프, 그 앞의 "진단" 단계를 건너뜀

- **실수 날짜**: 2026-06-02 (01-03 4-Phase 검증 자답)
- **상황**: §7 면접 질문 Q1·Q2·Q5 자답 중 *세 번 반복*된 패턴
- **잘못 답한 내용**:
  - Q1 — "자바는 네이티브 메모리를 많이 쓴다"(전제)에서 멈춤. *그래서 무슨 쓸모인가*(메시지→영역 진단)를 안 닫음
  - Q2 — 의심 영역(네이티브·스레드 과다)까지는 맞췄으나 *무슨 도구로 진단하는가*(NMT)를 안 붙임
  - Q5 — "물리 메모리를 늘리는 게 가장 좋아 보인다"로 단정. *누수인지 부족인지 가리는 진단 단계*를 건너뜀
- **정답**: 답의 공통 골격은 **"진단(어느 영역·누수인가 부족인가)이 처방(증설·옵션·코드 수정)보다 먼저"**.
  - 면접 답변은 *전제·의심*에서 끝내지 않고 *행동·도구·결론*으로 닫는다 ("의심된다" → "그래서 NMT로 확인한다").
  - OOM이 *누수* 때문이면 물리 메모리를 늘려봐야 터지는 시점만 미뤄질 뿐이다. 힙 덤프·NMT로 누수/부족을 가린 뒤 처방한다.
  - 다이렉트 메모리는 물리 RAM이 아무리 많아도 `-XX:MaxDirectMemorySize` 한계에 닿으면 OOM이라, 증설로 안 막힌다.
- **참고 챕터**: 01-03 §1(힙 OOM 누수 vs 부족 표), §4(NMT 진단), §5(`-Xmx` 통제 영역 vs 비통제 영역), §7·정답 1·2·5
- **재방문 트리거**: 다음 복습 회차에 Q1·Q2·Q5를 다시 자답하되, *답하기 전에 "먼저 어떻게 확인하지?"를 한 문장 앞에 붙이는* 연습. 진단 단계가 자동으로 앞서 나오면 졸업

## 패턴 2 — "재현용 세팅 의도"와 "운영 튜닝 판단"을 섞어 읽음

- **실수 날짜**: 2026-06-02 (01-03 4-Phase Phase 1)
- **상황**: §2.1 SOF 코드가 `-Xss`를 *작게* 잡은 이유를 물었을 때
- **잘못 답한 내용**: "스택 크기를 늘리는 건 비용 대비 효용이 없다 / 힙은 키워서 GC를 줄이는 게 좋다" — *운영 튜닝* 관점으로 답함
- **정답**: 질문은 *재현 코드가 OOM을 빨리 터뜨리려는 실험 세팅*에 대한 것이었다. SOF는 *한 스레드 안에서 빨리 넘치게* 하려고 `-Xss`를 작게, native-thread OOM은 *스레드당 네이티브를 빨리 소진*시키려고 크게 잡는다. 같은 옵션이 반대 방향인 이유 = 터뜨리는 메커니즘이 *깊이* vs *개수*로 정반대이기 때문. *재현은 빨리 터뜨리는 게 목적, 운영은 안 터지게 하는 게 목적* — 정반대 목적이라 세팅도 반대로 읽는다.
- **참고 챕터**: 01-03 §2.1(보강된 "빨리 터뜨리려는 실험 세팅" 설명), §2.2
- **재방문 트리거**: 옵션 값을 볼 때 "이게 *재현* 세팅인가 *운영* 세팅인가"를 먼저 구분하는 습관. 이 구분이 자동이면 졸업

## 패턴 3 — "참조가 네이티브로 넘어갔다"는 이사 방향 혼동

- **실수 날짜**: 2026-06-02 (01-03 4-Phase Phase 1, Q4 자답)
- **상황**: JDK 7 문자열 상수 풀 변경을 설명할 때
- **잘못 답한 내용**: "JDK 7부터 실체화 값은 힙에 남고 *참조값만 네이티브 영역으로 넘어갔다*" — 참조가 이사 간 것으로 표현
- **정답**: 이사 간 건 *실체*(문자열 객체 → 자바 힙)다. *참조 등록부*(String Table)는 원래부터 메서드 영역에 있었고 JDK 7 이후에도 거기 남았다. 참조는 움직이지 않았다. "실체가 힙으로 이사, 참조는 메서드 영역에 잔류"가 정확한 방향.
- **참고 챕터**: 01-03 §3.1(도서관 비유 — 실제 책=실체, 색인 카드=참조), 정답 4
- **재방문 트리거**: "무엇이 어디로 *이사*했는가"를 화살표로 그려보기 — 실체→힙(이동), 참조→메서드 영역(잔류). 방향이 입에 붙으면 졸업
