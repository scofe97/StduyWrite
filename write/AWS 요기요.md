# AWS 요기요 

## 클라우드 운영

클라우드 도입 후 비즈니스가 확장되면서 관리할 서비스의 폭증

운영자의 하루

- 콘솔전환 -> 지표 확인 -> 로그 검색 -> 원인 추론 -> 조치
- 이슈대응의 병목은 사람이 상황을 파악하는 시간

### AIOps

AIOps는 모니터링 자동화가 아닌 AI가 판단까지 수행하는 것

AIOps (관찰 - 이해 - 행동)

- AI가 이상 탐지 + 원인 분석 + 조치 제안
- 자연어 질의로 즉시 답변
- AI가 주기적으로 인텔리전스 리포트 생성

Amazon Bedrock 코어

AI 에이전트를 프로덕션 환경에 안전하게 배포하고 편리학 운영하기 위한 플랫폼

- 에이전트 코어 런타임 / 아이덴티티 / 메모리 / 게이트웨이
- 서버리스 / 시큐리티(에이전트 별 독립 보안) / 오케스트레이션 (멀티 에이전트 네이티브 지원)

통합 모니터링

- 서비스 영향도 분석
- 리소스 최적화
- 이벤트 리포트 자동화
