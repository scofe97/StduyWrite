# 쿼리 확인

Predicate

BooleanExpression - Predicate의 하위 버전

JPQLQueryy<T>

- Select().from().where().orderby()로 빌드 중인 쿼리 본체다.
- 마지막 fetch() / fetchone()으로 단건 받음

PathBuuilder<T> 

- 엔티티의 경로를 동적으로 가리키는 도구

- 같은 엔티티를 2번 별칭 다르게 JOIN하거나 서브쿼리 별도 별칭 쓰기 위함

  





```java
public int selectAprvMngCount(SelectAprvMngListQuery request) {
  	// 
    ResolvedListQuery<ApprovalManagementListColumn> resolved = resolveListQuery(LIST_QUERY_RESOLVER, request);
  
    // 
    ApprovalManagementListQueryContext ctx = defaultQueryContext();
    Predicate predicate = buildPredicate(resolved, ctx, latestVersionPredicate(ctx));

    Long count = applyJoins(
        queryFactory.select(ctx.approvalBasicId().getString("atrzId").countDistinct())
            .from(ctx.approvalBasic()),
        ctx
    ).where(predicate).fetchOne();

    return count == null ? 0 : count.intValue();
}
```

SELECT COUNT(DISTINCT approvalBasic.atrzId)
FROM TB_TPS_AZ_001 approvalBasic
LEFT JOIN TB_TPS_CM_001 mdfrUser
       ON mdfrUser.userId = approvalBasic.mdfrId
WHERE approvalBasic.delYn = 'N'
  AND approvalBasic.atrzSeCd <> '03'
  AND approvalBasic.vsrn = (
        SELECT MAX(sub.vsrn)
        FROM TB_TPS_AZ_001 sub
        WHERE sub.atrzId = approvalBasic.atrzId
          AND sub.delYn = 'N'
      )
  -- (검색 키워드가 있으면 OR 9개가 여기 한 덩어리로 추가)
  -- (상세 조건이 있으면 각각 AND 로 추가)
