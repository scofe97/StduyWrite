### [Ktkit: A Kotlin toolkit for building server applications with Ktor](https://github.com/smyrgeorge/ktkit)
- **소스**: HN
- **관련 키워드**: Java/Kotlin, Message Queue, Spring Boot 대안
- **요약**: Ktor 기반 Kotlin 서버 개발을 위한 멀티플랫폼 툴킷. DI, 라우팅, 에러 처리, DB 접근을 통합하며 PGMQ를 통한 PostgreSQL 기반 메시지 큐잉도 지원한다. Spring Boot의 경량 대안으로 포지셔닝된다.
