# Runners High Project

멀티 모듈 프로젝트 저장소

## 구조

```
project/
├── docs/           # 프로젝트 기획 및 기술 문서
│   └── TPS/        # TPS 프로젝트 문서
│
├── docker/         # Docker 관련 패키지
├── git-api/        # Git API 패키지
├── git-provider/   # Git Provider 패키지
└── tps-api/        # TPS API 패키지
```

## 패키지

| 패키지 | 설명 |
|--------|------|
| docker | Docker 설정 및 컨테이너 관리 |
| git-api | Git 작업 API |
| git-provider | Git 제공자 통합 (GitHub, GitLab 등) |
| tps-api | TPS 백엔드 API |
