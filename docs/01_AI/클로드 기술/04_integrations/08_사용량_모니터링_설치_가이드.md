# Claude Code 사용량 모니터링 설치 가이드

Claude Code CLI 하단 상태바에 실시간 사용량을 표시하는 방법입니다.

## 도구 선택 가이드

| 목적 | 추천 도구 |
|------|----------|
| 한도 관리 (5시간 블록, 주간 한도) | **claude-limitline** |
| 비용 추적 (달러 단위) | ccusage |
| 완전 커스터마이징 | ccstatusline |

---

## 1. claude-limitline (추천)

사용량 한도와 리셋 시간을 직관적으로 표시하는 도구입니다.

### 표시 정보 해석

```
◫ 42% (1h31m)  ○ 85% (wk 99%)  ◐ 50%
```

| 표시 | 의미 |
|------|------|
| `◫ 42% (1h31m)` | 5시간 블록 42% 사용, 리셋까지 1시간 31분 |
| `○ 85%` | Opus 주간 사용량 85% |
| `(wk 99%)` | 주간 사이클 99% 진행 (곧 리셋) |
| `◐ 50%` | Sonnet 주간 사용량 50% |

### 아이콘 범례

| 아이콘 | 모델 |
|--------|------|
| `◫` | 전체 블록 |
| `○` | Opus |
| `◐` | Sonnet |
| `◇` | Haiku |

### 설치 방법

`~/.claude/settings.json`에 추가:

```json
{
  "statusLine": {
    "type": "command",
    "command": "npx -y claude-limitline@latest",
    "padding": 0
  }
}
```

### 커스텀 설정

설정 파일: `~/.claude/claude-limitline.json`

```json
{
  "display": {
    "style": "powerline",
    "useNerdFonts": true,
    "compactMode": "auto",
    "compactWidth": 80
  },
  "directory": { "enabled": false },
  "git": { "enabled": false },
  "model": { "enabled": true },
  "block": {
    "enabled": true,
    "displayStyle": "text",
    "barWidth": 10,
    "showTimeRemaining": true
  },
  "weekly": {
    "enabled": true,
    "displayStyle": "text",
    "barWidth": 10,
    "showWeekProgress": true,
    "viewMode": "smart"
  },
  "budget": {
    "pollInterval": 15,
    "warningThreshold": 80
  },
  "theme": "dark",
  "segmentOrder": ["model", "block", "weekly"],
  "showTrend": true
}
```

### 설정 옵션 상세

#### 디스플레이 설정

| 옵션 | 설명 | 값 |
|------|------|-----|
| `display.useNerdFonts` | Nerd Font 아이콘 사용 | `true` / `false` |
| `display.compactMode` | 컴팩트 모드 | `"auto"`, `"always"`, `"never"` |
| `display.compactWidth` | 컴팩트 트리거 너비 | 숫자 (기본: 80) |

#### 세그먼트 on/off

| 세그먼트 | 설명 | 기본값 |
|---------|------|--------|
| `directory` | 프로젝트/리포지토리명 | `true` |
| `git` | 브랜치명 + 더티 표시(●) | `true` |
| `model` | Claude 모델명 | `true` |
| `block` | 5시간 블록 사용량 | `true` |
| `weekly` | 7일 주간 사용량 | `true` |

#### 블록/주간 표시 스타일

| 옵션 | 설명 |
|------|------|
| `displayStyle: "text"` | 텍스트로 표시 (예: `42%`) |
| `displayStyle: "bar"` | 프로그레스 바로 표시 |
| `barWidth` | 바 너비 (문자 수) |
| `showTimeRemaining` | 리셋 남은 시간 표시 |
| `showWeekProgress` | 주간 진행률 표시 |

#### 주간 표시 모드 (viewMode)

| 모드 | 설명 | 예시 |
|------|------|------|
| `simple` | 전체 주간 사용량만 | `○ 47% (wk 85%)` |
| `smart` | 현재 모델 + 전체 | `◇7% \| ○47%` |

#### 경고 설정

```json
"budget": {
  "pollInterval": 15,      // API 호출 간격 (분)
  "warningThreshold": 80   // 경고 색상 트리거 (%)
}
```

### 테마 옵션

| 테마 | 설명 |
|------|------|
| `dark` | 기본값, 따뜻한 갈색 + 청록색 |
| `light` | 밝은 배경용 |
| `nord` | Nord 팔레트 |
| `gruvbox` | Gruvbox 팔레트 |
| `tokyo-night` | Tokyo Night 팔레트 |
| `rose-pine` | Rosé Pine 팔레트 |

### 세그먼트 순서 변경

```json
"segmentOrder": ["directory", "git", "model", "block", "weekly"]
```

원하는 순서로 배열 수정.

### 색상 의미

| 색상 | 상태 |
|------|------|
| 청색/초록 | 정상 |
| 주황색 | 주의 (warningThreshold 이상) |
| 빨강 | 위험 (한도 임박) |

## 2. ccstatusline

완전 커스터마이징 가능한 상태바. 대화형 TUI로 설정합니다.

### 설치 및 설정

```bash
# 설정 UI 실행
npx ccstatusline@latest
```

`~/.claude/settings.json`:

```json
{
  "statusLine": {
    "type": "command",
    "command": "npx -y ccstatusline@latest",
    "padding": 0
  }
}
```

### 설정 파일

위치: `~/.config/ccstatusline/settings.json`

```json
{
  "version": 3,
  "lines": [
    [
      { "id": "1", "type": "model", "color": "cyan" },
      { "id": "2", "type": "separator" },
      { "id": "3", "type": "session-clock", "color": "green" },
      { "id": "4", "type": "separator" },
      { "id": "5", "type": "tokens-total", "color": "yellow" },
      { "id": "6", "type": "separator" },
      { "id": "7", "type": "context-length", "color": "brightBlack" }
    ]
  ],
  "colorLevel": 2,
  "powerline": { "enabled": false }
}
```

### 사용 가능한 위젯

| 위젯 | type | 설명 |
|------|------|------|
| 모델명 | `model` | 현재 Claude 모델 |
| 세션 시간 | `session-clock` | 세션 경과 시간 |
| 블록 타이머 | `block-timer` | 5시간 블록 경과 시간 |
| 토큰 (총합) | `tokens-total` | 총 토큰 사용량 |
| 토큰 (입력) | `tokens-input` | 입력 토큰 |
| 토큰 (출력) | `tokens-output` | 출력 토큰 |
| 컨텍스트 길이 | `context-length` | 컨텍스트 토큰 수 |
| 컨텍스트 % | `context-percentage` | 컨텍스트 사용률 |
| Git 브랜치 | `git-branch` | 현재 브랜치 |
| Git 변경 | `git-changes` | 변경사항 (+/-) |
| 구분자 | `separator` | 위젯 구분선 |
| 커스텀 텍스트 | `custom-text` | 사용자 정의 텍스트 |

### 색상 옵션

```
cyan, green, yellow, red, blue, magenta,
white, black, brightBlack, brightRed, etc.
```

### 커스텀 예시: 최소 구성

```json
{
  "version": 3,
  "lines": [
    [
      { "id": "1", "type": "model", "color": "cyan" },
      { "id": "2", "type": "separator" },
      { "id": "3", "type": "context-percentage", "color": "yellow" }
    ]
  ]
}
```

결과: `Model: Opus 4.5 | Ctx: 45%`

## 3. ccusage

토큰 사용량 및 비용 분석 도구.

### 상태바 설정

```json
{
  "statusLine": {
    "type": "command",
    "command": "npx -y ccusage@latest statusline",
    "padding": 0
  }
}
```

### CLI 명령어

```bash
# 실시간 모니터링 (5시간 블록)
npx ccusage@latest blocks --live

# 월별 리포트
npx ccusage@latest monthly

# 세션별 리포트
npx ccusage@latest sessions

# 모델별 비용 분석
npx ccusage@latest --breakdown
```

### 상태바 옵션

| 옵션 | 설명 |
|------|------|
| `--context-low-threshold 50` | 초록색 표시 기준 (%) |
| `--context-medium-threshold 80` | 노란색 표시 기준 (%) |
| `--visual-burn-rate emoji` | 소비율에 이모지 추가 |
| `--cost-source ccusage` | 비용 계산 방식 |

예시:
```json
{
  "statusLine": {
    "type": "command",
    "command": "npx -y ccusage@latest statusline --context-low-threshold 60 --context-medium-threshold 85",
    "padding": 0
  }
}
```

## Claude Code 사용량 체계

### 5시간 블록 (Rolling Window)

- 첫 메시지부터 5시간 동안 사용량 추적
- 5시간 후 자동 리셋
- 한도 도달 시 남은 시간만큼 대기 필요

### 7일 주간 한도

- 7일 롤링 윈도우로 사용량 관리
- 모델별(Opus/Sonnet) 별도 한도 적용
- `wk %`는 주간 사이클 진행률 (100%에서 리셋)
