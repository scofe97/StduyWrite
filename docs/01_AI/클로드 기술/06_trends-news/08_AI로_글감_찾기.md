## 문제 상황

글쓰기 초보자들의 공통 고민:

> "무엇을 써야 할지 모르겠다"

메모는 많지만 흩어져 있어서 주제로 연결되지 않음.

---

## 솔루션: Claude Code의 rawnotes 분석

### 핵심 인사이트

> "주제가 없었던 것이 아니라 흩어져 있었을 뿐"

AI를 활용해 산발적인 메모에서 숨겨진 패턴과 주제를 추출.

---

## 워크플로우

### Step 1: 데이터 수집

Notion에 저장된 메모들을 마크다운으로 내보내기:
- 약 100여 개의 메모
- 다양한 주제와 생각들

### Step 2: AI 분석

```bash
/extract-themes
```

**3분 만에 도출된 결과**:
- 5개 핵심 테마
- 15개 구체적 주제

### Step 3: 고급 분석

단순 나열을 넘어선 인사이트:

| 분석 유형 | 설명 |
|----------|------|
| **독특한 관점** | "당신만의 시각"이 무엇인지 도출 |
| **감정 곡선** | 두려움 → 호기심 → 전략적 채택 |
| **부족한 요소** | 콘텐츠에서 빠진 부분 지적 |

---

## 분석 결과 예시

### 감정 곡선 분석

```
두려움 (초기)
   ↓
호기심 (탐색)
   ↓
전략적 채택 (활용)
```

메모들의 시간적 흐름에서 감정 변화 패턴 발견.

### 테마 추출

```markdown
## 핵심 테마 5개
1. [테마 1]
2. [테마 2]
...

## 구체적 주제 15개
- 테마 1 관련: [주제 A], [주제 B], [주제 C]
- 테마 2 관련: [주제 D], [주제 E], [주제 F]
...
```


## 적용 포인트

### 1. 메모 시스템 구축

- Notion, Obsidian 등에 메모 축적
- 마크다운 형식으로 내보내기 가능하게 유지

### 2. 정기적 테마 추출

```bash
# 월 1회 또는 메모 50개마다
/extract-themes
```

### 3. 글쓰기 파이프라인

```
메모 축적 → 테마 추출 → 주제 선택 → 리서치 → 초안 작성
```

## 커스텀 스킬 아이디어

### /extract-themes 스킬

```yaml
---
name: extract-themes
description: "메모/노트에서 주제와 테마 추출. 글감 발굴, 콘텐츠 기획에 활용.
             트리거: '/extract-themes', '글감 찾기', '주제 추출', '테마 분석'"
---
```

**분석 항목**:
1. 핵심 테마 (3-5개)
2. 구체적 주제 (테마당 3개)
3. 독특한 관점 도출
4. 감정/시간 곡선
5. 부족한 요소 지적

## 핵심 메시지

> **AI의 역할**: 흩어진 메모들을 연결하여 숨겨진 패턴을 드러내는 것

글감이 없는 것이 아니라, 연결되지 않았을 뿐.
AI는 그 연결고리를 찾아주는 도구.
