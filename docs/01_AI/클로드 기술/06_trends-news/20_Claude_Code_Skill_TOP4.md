# Claude Code Skill 10만 개 중 진짜 살아남은 4개

> 1주일의 업무를 위한 도구 셋업에 쓴 주말 하루가 수백 시간의 반복 노동을 없앤다

---

## 배경

요즘 AI 코딩 에이전트용 Skill이 하루에도 수백 개씩 쏟아지고 있다. GitHub에 등록된 오픈소스 Skill만 10만 개가 넘는다.

솔직히 처음엔 좋아보이는 건 다 깔았는데, **90%는 설치하고 한 번도 안 쓰게 되더라.** 그래서 주말 동안 실제로 써온 것들 중 "이건 진짜 매일 쓴다" 싶은 4개만 추렸다.

---

## 1. Superpowers — AI가 코드부터 짜지 않게 만드는 개발 방법론

> GitHub Stars: 4.1만 | Skill 생태계 사실상 1위
> Repo: https://lnkd.in/gwas6su7

단순한 코드 생성 보조가 아니라, **에이전트의 소프트웨어 개발 프로세스 자체를 바꿔놓는 프레임워크**다.

### 7단계 워크플로우

```
브레인스토밍 → 계획 수립 → TDD → 코딩 → 코드 리뷰 → Git Worktree 관리 → 완료
```

### 핵심 특징

| 특징 | 설명 |
|------|------|
| **설계 우선** | 설치하면 에이전트가 먼저 "뭘 만들려는 건지" 질문, 설계 문서 작성 후 코딩 시작 |
| **작업 분할** | 2~5분 단위로 쪼개서 독립된 하위 에이전트에 배분 |
| **자동 리뷰** | 2단계 자동 리뷰 수행 |

> 직접 써보고 가장 놀랐던 건 Claude Code가 정말 오랜 시간 자율적으로 작업하면서도 **계획에서 벗어나지 않는다**는 점이었다.

```bash
npx skills add obra/superpowers
```

## 2. Humanizer — AI가 쓴 글에서 "AI 냄새"를 24가지로 잡아내는 필터

> Repo: https://lnkd.in/g8YGmeG9

위키피디아 Wiki Project AI Cleanup 팀이 수천 건의 AI 생성 텍스트를 분석해서 만든 패턴을 기반으로 동작한다.

### 24개 패턴 체크리스트 (예시)

| 패턴 | 예시 |
|------|------|
| **AI 특유 어휘** | "testament", "landscape", "showcasing" |
| **의미 부풀리기** | "혁신적인", "획기적인" 남발 |
| **세 단어 나열** | "빠르고, 안전하고, 효율적인" |
| **엠대시 남발** | — 를 과도하게 사용 |
| **이모지 오남용** | 문맥 없는 이모지 삽입 |

- 수정 전후 예시가 패턴마다 포함되어 있어서 왜 고쳐야 하는지 바로 이해 가능
- AI로 초안 잡고 그대로 올리면 읽는 사람이 첫 문장에서 알아챈다
- 다만, **한국어에 맞게 적용할 필요**가 있다

```bash
npx skills add blader/humanizer
```

## 3. UI/UX Pro Max — AI가 만든 화면에서 템플릿 티를 없애주는 디자인 엔진

> Repo: https://lnkd.in/gdNk58JW

67가지 UI 스타일, 96개 색상 팔레트, 57개 폰트 조합, 100개 산업별 추론 규칙을 통째로 내장한 Skill이다.

### 내장 규칙

| 항목 | 수량 |
|------|------|
| UI 스타일 | 67가지 |
| 색상 팔레트 | 96개 |
| 폰트 조합 | 57개 |
| 산업별 추론 규칙 | 100개 |
| 기술 스택 가이드 | 13개 (React, Next.js, Vue, SwiftUI, Flutter 등) |

### 핵심: 안티패턴 검출

가장 큰 차이를 느낀 건 안티패턴 검출이다. AI가 만든 UI에서 가장 티나는 패턴을 정확히 잡아내고 대안을 제시해준다:

- 보라+분홍 그라디언트
- 이모지 아이콘
- 기본 템플릿 레이아웃

> "뷰티 스파 랜딩페이지 만들어줘" 한 마디면 업종 맞춤 디자인 시스템 전체를 자동 생성하고, **"이 업종에서 하면 안 되는 것"** 목록까지 제공한다.

```bash
npx skills add nextlevelbuilder/ui-ux-pro-max-skill
```

## 4. Find Skills (Vercel 공식) — 10만 Skill 생태계를 한 줄로 검색하고 설치

> Repo: https://lnkd.in/g4RS6dTJ
> 웹 검색: https://skills.sh/

Vercel Labs에서 만든 에이전트 Skill 전용 CLI다. Skill이 많아질수록 진가를 발휘하는 관리 도구다.

### 핵심 명령어

| 명령어 | 기능 |
|--------|------|
| `npx skills find <키워드>` | 키워드 검색 |
| `npx skills add <패키지>` | 즉시 설치 |
| `npx skills check` | 업데이트 확인 |
| `npx skills update` | 일괄 갱신 |

- Claude Code, Codex, Cursor, Gemini CLI, Windsurf 등 **27개 에이전트 동시 지원**
- Skill 설치, 검색, 업데이트를 전부 해결

```bash
npx skills add vercel-labs/skills
```

## 정리

| # | Skill | 핵심 가치 | 한 마디 |
|---|-------|-----------|---------|
| 1 | **Superpowers** | 개발 프로세스 강제 | 코드부터 짜지 않게 만든다 |
| 2 | **Humanizer** | AI 텍스트 필터 | 24가지 "AI 냄새" 제거 |
| 3 | **UI/UX Pro Max** | 디자인 엔진 | 템플릿 티를 없앤다 |
| 4 | **Find Skills** | Skill 관리 CLI | 10만 개에서 원하는 걸 찾는다 |

### 1인 유니콘의 작업 환경

모니터 여러 대에 터미널을 동시에 띄워놓고 AI 에이전트와 병렬로 작업하는 모습. 한쪽에선 코드, 다른 쪽에선 디자인, 또 다른 화면에선 문서. **실제 업무를 수행하는 건 전부 에이전트이고, 사람은 관제탑 역할만 한다.**

혼자서 개발, 디자인, 글쓰기, 배포를 동시에 돌리려면 이런 Skill 세팅에 시간을 아끼면 안 된다.
