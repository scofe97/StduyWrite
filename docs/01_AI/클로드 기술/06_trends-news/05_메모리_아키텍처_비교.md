# Claude vs ChatGPT 메모리 아키텍처 비교

> 출처: https://news.hada.io/topic?id=23045

## 핵심 개념

두 AI 서비스는 **완전히 상반된 메모리 시스템**을 채택하며, 각각의 타겟 사용자층과 제품 철학을 반영

---

## 메모리 방식 비교

### Claude

| 특징 | 설명 |
|------|------|
| **명시적 호출** | "이전에 이야기한 내용 알려줘" 같은 직접 요청 시에만 작동 |
| **실시간 검색** | AI 요약본이 아닌 실제 대화 이력을 키워드 검색 |
| **도구 활용** | `conversation_search` (주제별), `recent_chats` (시간별) |

### ChatGPT

| 특징 | 설명 |
|------|------|
| **자동 불러오기** | 매 대화마다 사용자 프로필과 히스토리 자동 로드 |
| **즉각적 개인화** | 별도 요청 없이 맥락 반영 |

---

## 철학적 차이

| 구분 | Claude | ChatGPT |
|------|--------|---------|
| **타겟 사용자** | 개발자, 전문가 | 대중 시장 소비자 |
| **핵심 가치** | 투명성, 제어, 개인정보 보호 | 편의성, 자동화 |
| **접근 방식** | 사용자 주도 | 시스템 주도 |

---

## 최신 동향

Anthropic은 팀/엔터프라이즈 계정에 ChatGPT와 유사한 **자동 메모리 기능** 도입 중
