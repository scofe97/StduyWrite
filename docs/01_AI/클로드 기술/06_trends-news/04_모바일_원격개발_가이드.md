# 모바일 Claude Code 원격 개발 가이드

> 출처:
> - https://news.hada.io/topic?id=25639 (Doom Coding)
> - https://news.hada.io/topic?id=25578 (Claude Code On-The-Go)

## 핵심 개념

**Doom Coding**: 무한 스크롤링 대신 스마트폰으로 원격 개발
**Claude Code On-The-Go**: 모바일에서 다중 에이전트 병렬 실행

---

## 기본 구성 (Doom Coding)

### 필수 준비물
- 24시간 가동 인터넷 연결 컴퓨터
- 스마트폰
- Claude Pro 구독

### 3단계 설정

| 단계 | 작업 |
|------|------|
| 1. 컴퓨터 | Tailscale 설치, SSH 활성화, Claude Code 설치 |
| 2. 스마트폰 | Tailscale 앱 + Termius SSH 클라이언트 설치 |
| 3. 연결 | VPN 활성화 → 터미널 접속 → `claude` 실행 |

---

## 고급 구성 (On-The-Go)

### 인프라 구성
- **Vultr VM**: 시간당 $0.29, 실리콘밸리 배치
- **Tailscale VPN**: 안전한 접근
- **Termius + mosh**: 모바일 SSH 연결 유지
- **보안**: nftables, fail2ban 등 다중 계층

### 핵심 기술

| 기술 | 용도 |
|------|------|
| **tmux 세션** | 여러 에이전트 병렬 실행 (최대 6개) |
| **Git worktree** | 기능별 독립 브랜치 관리 |
| **Poke 웹훅** | 푸시 알림 시스템 |
| **PreToolUse hook** | Claude 질문 감지 |

### 비동기 개발 워크플로우

```
1. Claude에게 작업 지시 (10~20분 소요 작업)
2. 사용자는 다른 일 진행
3. 알림 수신 시에만 응답
4. PR 검토, 리팩터링, 버그 수정 이동 중 수행
```

## 실천 방법

### 환경 설정
```bash
# .zshrc에 자동 tmux 연결 설정
if [ -z "$TMUX" ]; then
  tmux attach || tmux new
fi
```

### 포트 할당
- 브랜치명 해시 기반 자동 배정

### 비용 관리
- 일일 최대 비용 제한 설정
- Vultr: 시간당 $0.29 수준

## 장점

- 노트북 없이도 완전한 개발 주기 실현
- 이동 시간 활용 (출퇴근, 대기 시간)
- 무한 스크롤 → 생산적 시간 전환
