# Anthropic의 각 팀들이 Claude Code를 활용하는 방법

> 출처: https://news.hada.io/topic?id=22194

## 개요

Anthropic 내부 10개 부서에서 Claude Code를 활용하여 **2~4배 시간 단축** 효과를 경험. 개발자와 비개발자 모두 사용 가능.

---

## 부서별 활용 사례

### 1. 데이터 인프라팀
- Kubernetes 장애 진단 및 해결 자동화
- 비개발자용 평문 워크플로 변환
- 신입 온보딩 시간 대폭 단축

### 2. 제품개발팀
- "자동수용 모드"로 프로토타입 80% 완성 후 최종 조정
- 실시간 협업 코딩
- 자동화된 테스트 생성

### 3. 보안팀
- Terraform 코드 보안 리뷰 신속화
- 인프라 문제 5분 이내 해결
- 문서/런북 자동 생성

### 4. 추론팀
- 코드베이스 이해 시간 **80% 절감**
- 엣지 케이스 포함 테스트 자동 생성
- 다국어 코드 변환

### 5. 데이터사이언스/ML팀
- JavaScript 미숙해도 React 대시보드 구축
- 반복 리팩토링 자동화 (30분 완전 자동 처리)
- 언어 장벽 제거

### 6. 성장마케팅팀
- Google Ads 카피: **2시간 → 15분** 단축
- Figma/Meta Ads 연동 자동화
- 크리에이티브 생성 **10배 증가**

### 7. 제품디자인팀
- 디자이너가 직접 UI 개선/상태관리 구현
- GitHub Actions 기반 자동 코드 제안
- 복잡 협업 프로젝트: **1주 → 1시간**

### 8. RL엔지니어링팀
- 중소 기능은 "자율+감독" 방식으로 처리
- 테스트/코드리뷰 자동화
- 중소형 PR 약 **33%** 한 번에 완성

### 9. 법무팀
- 접근성 솔루션 직접 커스터마이징
- 워크플로 자동화 프로토타입 제작
- MCP 연동 보안/컴플라이언스 체크

### 10. 제품엔지니어링팀
- 미지의 코드베이스 독립 작업 가능
- 컨텍스트 전환 부담 제거
- 로테이션 엔지니어 온보딩 가속화

---

## Best Practices

| 방법 | 설명 |
|------|------|
| **Claude.md 문서화** | 컨텍스트, 규칙, 사례 상세 기록 |
| **슬롯머신 방식** | 70-80% 완성도 받아들이고 재시도 반복 |
| **체크포인트 저장** | 30분 작업 단위로 상태 저장/롤백 |
| **구체적 프롬프트** | 추상적 지시 대신 명확한 요구사항 전달 |
| **이미지/스크린샷** | 시각적 피드백으로 UI/프로토타입 빠르게 개선 |
| **병렬 세션 운영** | 프로젝트별 독립 인스턴스로 컨텍스트 보존 |

---

## 연동 도구

- **MCP 서버**: 민감 데이터 안전 처리
- **Figma 플러그인**: 대량 크리에이티브 자동 생성
- **Google Ads/Meta Ads API**: 마케팅 자동화
- **GitHub Actions**: 코드 제안/티켓팅 자동화
- **Kubernetes CLI**: 인프라 관리
