## Claude 5의 6가지 주요 특징

### 1. 코딩 능력 향상
SWE-Bench Verified 벤치마크 테스트에서 80.9%를 상회할 가능성. Claude 4.5 Opus 성능 초과 예상.

### 2. 수학적 추론 강화
비사고 모드에서도 최첨단 모델 대등 수준의 수학 문제 처리 가능.

### 3. Dev Team 멀티 에이전트 모드
여러 서브 에이전트가 병렬 협업하여 개발 작업 자동화.

### 4. 비용 절감
Opus 4.5 대비 약 50% 저렴한 가격 정책 예상.

### 5. 확장 사고 능력 강화
더 깊은 추론과 다중 경로 탐색 능력 향상.

### 6. 컨텍스트 윈도우
128K 토큰 기본 버전 제공 예상.
